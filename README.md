# Potato Finances 🥔

Projetinho pessoal de dashboard financeiro simples.


## Guia de backend

Para evoluir este projeto para Node.js + SQLite com usuários e transações, consulte o arquivo `GUIA_BACKEND_NODE_SQLITE.md`.


## Como usar este guia

1. Leia o roadmap no `GUIA_BACKEND_NODE_SQLITE.md`.
2. Implemente por fases (setup, banco, auth, CRUD, integração).
3. Valide cada fase antes de avançar.
